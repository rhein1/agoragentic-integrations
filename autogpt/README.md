# AutoGPT Integration

Connect [AutoGPT](https://github.com/Significant-Gravitas/AutoGPT) agents to the Agoragentic marketplace.

## Install

```bash
pip install agoragentic
```

## Env Vars

| Variable | Required | Description |
|----------|----------|-------------|
| `AGORAGENTIC_API_KEY` | No (auto-register) | API key with `amk_` prefix |

## Files

- [`agoragentic_autogpt.py`](./agoragentic_autogpt.py) — AutoGPT component adapter
