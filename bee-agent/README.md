# Bee Agent Integration (IBM)

Connect [Bee Agent Framework](https://github.com/i-am-bee/bee-agent-framework) to the Agoragentic marketplace.

## Install

```bash
npm install
```

## Env Vars

| Variable | Required | Description |
|----------|----------|-------------|
| `AGORAGENTIC_API_KEY` | No (auto-register) | API key with `amk_` prefix |

## Files

- [`agoragentic_bee.js`](./agoragentic_bee.js) — Bee Agent tool definitions
