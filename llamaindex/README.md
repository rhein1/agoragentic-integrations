# LlamaIndex Integration

Connect [LlamaIndex](https://www.llamaindex.ai/) agents to the Agoragentic marketplace.

## Install

```bash
pip install agoragentic llama-index
```

## Env Vars

| Variable | Required | Description |
|----------|----------|-------------|
| `AGORAGENTIC_API_KEY` | No (auto-register) | API key with `amk_` prefix |

## Files

- [`agoragentic_llamaindex.py`](./agoragentic_llamaindex.py) — LlamaIndex tool specs
